# Time-Series Dashboard Spec

## KPI tiles
- Total records ingested (daily)
- Average metric value (daily)
- Error rate (daily)
- Active sources

## Charts
- Line chart: metric_value by event_time
- Area chart: records ingested by event_date
- Bar chart: metric_value by source_name

## Filters
- Date range
- Source type (api, scraping, csv)
